### Notebooks and code for the paper: "Multi-scale bathymetric modulation of coastal trapped waves generated in Sebastián Vizcaíno Bay, Baja California"

A. Thelandersson, K. Ramos-Musalem, B. D. Cornuelle, S. T. Gille and M. R. Mazloff

This repository is divided into:

createfigures/ notebooks and executable code used to generate figures in the paper

postprocessing/ code used to calculate the variables used in the analysis

datafiles/ data used to create the figures, generated by the code in postprocessing

All the python libraries needed to run the scripts and notebooks are listed in the the yaml file SVB.yml for the conda envirnoment SVB.
